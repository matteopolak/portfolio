title: Twitch Chat Downloader
date: 2022-11-01
tags: [prisma, rust, twitch]
repository: https://github.com/matteopolak/tcd
---

This tool allows users to download or archive chat messages from any Twitch live stream that aired in the last 30 days. It can download large amounts of chat messages quickly, with the capability of downloading up to 5,000,000 messages per hour. It includes features like multi-threading and connection pooling for efficient data retrieval.
